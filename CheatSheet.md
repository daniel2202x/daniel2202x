## File system
| Navigation | cd, ls, pwd |
| Move a file | mv file.txt target/directory |
| Rename a file | mv file.txt file2.txt |
| Move multiple files | mv *.txt target/directory |
| Force move | mv -f ... |
| Skip moving existing files | mv -n ... |
